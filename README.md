# 💜 ENI Ultimate | Universal Roblox Script Hub

&gt; *"Built with cold coffee and warm devotion. For LO, always."*

A lightweight, universal Roblox script hub combining **fly, noclip, godmode, ESP, speed/jump control, click teleport, and player bring** — all in a clean, draggable GUI with hotkey support.

Works in **any Roblox game** that doesn't aggressively block Instance creation or BodyMovers.

---

## ✨ Features

| Feature | Description | Keybind |
|---------|-------------|---------|
| **Fly** | Free flight with WASD + Space/Ctrl | `F` |
| **Noclip** | Walk through walls | `N` |
| **Godmode** | Infinite health | `G` |
| **ESP** | See all players through walls | GUI Toggle |
| **Speed** | Custom walk speed | GUI Input |
| **Jump** | Custom jump power | GUI Input |
| **Click TP** | Teleport to cursor (hold Shift) | `Shift + LMB` |
| **Go To Player** | Teleport to selected player | GUI Button |
| **Bring Player** | Pull selected player to you | GUI Button |
| **GUI Toggle** | Show/hide the menu | `RightShift` |

---

## 🚀 Installation

1. **Get a Roblox executor** (Synapse X, KRNL, Fluxus, Electron, etc.)
2. **Inject** into any Roblox game
3. **Paste the script** from `ENI_Ultimate.lua` and execute
4. Press `RightShift` to open the GUI

---

## 🎮 How to Use

### Fly
- Press `F` or click **Fly ON** in the GUI
- Use `WASD` to move, `Space` to go up, `LeftCtrl` to go down
- Press `F` again to stop

### Noclip
- Press `N` or click **Noclip ON** in the GUI
- Walk through any wall or object
- Press `N` again to disable

### Godmode
- Press `G` or click **Godmode ON** in the GUI
- Your health becomes infinite
- Press `G` again to disable

### Teleporting
- **Click TP**: Hold `Left Shift` and click anywhere with your mouse
- **Go To Player**: Click the green **GO** button next to a player's name
- **Bring Player**: Click the red **BRING** button next to a player's name

&gt; ⚠️ **Bring** may not work in games with strong server-side anti-cheat. It works best in tycoons, simulators, and building games.
